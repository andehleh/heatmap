# Ultra 2026 Heatmap 🎧

A live, shareable festival heatmap. Everyone opens the same link, enters their name, and picks their sets. Cells get warmer as more people vote — popular artists glow orange/red.

## Setup

1. Add your Firebase credentials to `lib/firebase.js`
2. Push to GitHub
3. Deploy on Vercel

See the setup guide from the chat for detailed steps.

## Admin

Log in as "Andy" to see the "manage" link for deleting members.
